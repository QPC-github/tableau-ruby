# TableauRuby

Unofficial Tableau API Ruby client

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tableau_ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tableau_ruby
